# Usage
```bash
git clone https://github.com/haturatu/quadrophenia.git
cd quadrophenia

```

## なにこれ
自分用にサーバで自己ホストしているWeb Appsにアクセスしやすいようしたフロントエンドです。
使う方いれば、パブリックに公開されているとよろしくないのでPort 4567番がパブリックにOpenではないこと確認して使ってください。
